GitHub Cloner

Genera script per scaricare i tuoi repository.

Necessario per repository privati, login via Google o per evitare limiti API.

Istruzioni Rapide

  1. Inserisci il tuo Username GitHub.
  2. Accedi con Google? La password Google non funziona.
    • Vai su Settings → Tokens (Classic)
    • Genera token (spunta repo)
    • Incollalo qui sopra
  3. Seleziona i repository e scarica lo script.
